【Audio File Mode】 音楽ファイルを読み込んで練習するモードです。 基本操作
  • ABループ: 波形をドラッグして範囲を指定します。
  • カウントイン: timer 有効時、再生前に4拍のクリックが流れます。
  • スピードアップ: bolt 有効時、ループを終えるごとに速度が0.05ずつ上がります（元の速度まで）。
録音機能（Premium）
  • マイクボタンで録音開始。自分の演奏を録音して、原曲と同時に再生してチェックできます。
  • S (Solo): 有効にすると原曲をミュートし、自分の録音のみを確認できます。
【YouTube Mode】 YouTube動画を利用して練習するモードです。 ループ設定
  • 「SET START」「SET END」で範囲を指定し、「LOOP OFF」ボタンを押してONに切り替えるとループ再生が始まります。
  • 保存ボタンで、現在の範囲をバッジとして保存できます。
履歴
  • 読み込んだ動画は履歴として保存され、次回から簡単にアクセスできます。
